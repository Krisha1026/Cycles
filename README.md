
# 🚴‍♀️ CityCycle Rentals - Smart Urban Mobility Solution

 [![Android](https://img.shields.io/badge/Android-8.0+-brightgreen?logo=android)](https://www.android.com)
  [![Java](https://img.shields.io/badge/Java-11+-orange?logo=java&logoColor=white)](https://java.com)

CityCycle Rentals is a full-featured **bike-sharing Android application** developed to offer a seamless, eco-friendly, and smart transportation solution for urban commuters. Designed with usability, flexibility, and sustainability in mind, this app delivers an intuitive bike rental experience right at your fingertips.

---

## 🌟 Key Features

### 1️⃣ User-Centric Experience
- 🔐 **Secure Authentication**  
  - Register/Login with validation  
  - Password encryption and secure data handling  

- 🙋‍♂️ **Personalized Profiles**  
  - Edit profile details (Name, Email, Password)  
  - View and manage account preferences  

- 🕓 **Rental History Tracking**  
  - Access previous ride history  
  - Track timestamps and total pricing  

---

### 2️⃣ Smart Bike Rental System
- 📍 **Bike Location & Reservation**  
  - View nearby docking stations  
  - Check bike availability in real-time  
  - Reserve your bike instantly  

- 🛠️ **Rental Interface**  
  - Start/End rental with a simple tap  
  - Duration-based rental with live updates  
  - View active rental status (Bike ID, Time used)

---

### 3️⃣ Flexible Payment Options
- 💳 **Multiple Payment Methods**  
  - Credit Card, PayPal, etc.  

- 💰 **Transparent Pricing Plans**  
  - **Pay-as-you-go:** First 30 minutes **FREE**  
  - **Daily/Monthly Memberships**  
  - Special offers, promotions, and discounts  

---

### 4️⃣ Enhanced User Support
- 🗣️ **Feedback System**  
  - Submit ratings and comments for services  

- ☎️ **Direct Support Access**  
  - Contact support via app interface  

- 📢 **Clear Promotions & Pricing Info**  
  - View all deals, rates, and terms from a dedicated section  

---

## 🛠️ Technical Implementation

| Feature | Tech Stack |
|--------|-------------|
| Language | Java |
| IDE | Android Studio |
| Database | SQLite (local storage) |
| UI | XML-based Android layouts |
| Architecture | MVC Pattern (Model-View-Controller) |
